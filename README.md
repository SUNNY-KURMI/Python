## 📋 Contents

### timer.py
A simple countdown timer application that runs in the terminal.

**Features:**
- Accepts time input in seconds
- Displays countdown in HH:MM:SS format
- Updates every second
- Displays "TIME'S UP!" when completed

**Usage:**
```bash
python timer.py
```

Then enter the desired time in seconds when prompted.

**Example:**
```
Enter your time in seconds: 3661
01:01:01
01:01:00
...
00:00:01
TIME'S UP!
```
